## Sobre o projeto
Este repositorio implementa o banco de dados para o sistema Hortifruti da Família. O sistema visa gerenciar o fluxo operacional de um hortifruti, cobrindo desde a cadeia de suprimentos (fornecedores e entradas de estoque), controle de perdas, categorização de produtos, até o fluxo de frente de caixa (PDV), envolvendo clientes, vendedores, caixas, operações de caixa, pedidos e múltiplos métodos de pagamento.
